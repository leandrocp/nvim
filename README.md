# Neovim Files

## Install
```
git clone git@github.com:leandrocp/nvim.git ~/.nvim

curl -fLo ~/.nvim/autoload/plug.vim --create-dirs https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim

cd ~/.nvim

bash install.sh
```
